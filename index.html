    • C language

    C is often called amiddle-levelcomputer language. This does not mean that C is less powerful language,harder to use,or less developed than a high-level language such as BASIC or Pascal,nor does it imply that C has the cumbersome nature of assembly language. As a middle-level language,C allows the manipulation of bits,bytes, and addresses- the basic elements with which the computer functions. Despite the fact ,C code is also very portable.portabilitymeans that it is easy adapt software written for one type of computer or operating system to another type.For example, if you can easily convert a program written for DOS so that it runs under Windows 2000, that program is portable.
